Describe the general direction you want the agent to push the project in. One short paragraph is plenty.
The agent re-reads this every pass and every task it picks must serve this goal — so keep it as the
north star, not a task list (tasks go in the backlog).

Example:
> Make the app a polished, fast single-page tool. Prioritize correctness and clear UX over new features;
> keep the bundle small and the code readable. Don't add a backend — everything stays client-side.

Copy this file to `GOAL.md` and replace the above with your own goal (or set it from the Workshop UI's
"General Goal" box, which writes the same file).
