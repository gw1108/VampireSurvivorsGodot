**Ebony Wings** is a [weapon](Weapon.md) in *[Vampire Survivors](Vampire_Survivors.md)*. It is the starting weapon of  [Exdash Exiviiq](Exdash_Exiviiq.md). It is unlocked by getting Peachone to Level 7.

Its [union](Union.md) is  [Vandalier](Vandalier.md), which also requires  [Peachone](Peachone.md) at max level.

## Effects {#Effects}

Ebony Wings is a bird that flies around the player and bombards a circling zone with projectiles. The zone revolves counterclockwise around the player. The bird moves behind the player at a slower speed, but the bombarding circle will always stay in a position relative to the player.

By default, Ebony Wings fires 4 sets of 4 projectiles. Each set fires all of its projectiles at about the same spot, set by the position of the visible zone at the instant that set starts. A set firing is on a different timer to the zone being active, so the first set can start firing up to a second after the zone lights up. This means that the number of sets per activation can vary by 1.

 [Duration](Duration.md) increases the time bombards are active, with 1 second roughly equal to 1 set by default.  [Area](Area.md) increases the Area-of-Effect of the projectiles, but not the size of the bombarding circle itself.  [Speed](Speed.md) increases how fast the circle moves.  [Amount](Amount.md) increases how many projectiles are fired per set.  [Cooldown](Cooldown.md) bonuses reduce the time between each activation, the time between each set of projectiles, and the time between each projectile within a set.  [Move Speed](Move_Speed.md) improves Ebony Wings\'s movement but has virtually no effect on its capability.

### Arcanas {#Arcanas}

-  [Gemini (I)](Gemini_(I).md) - Creates counterpart  [Zhar Ptytsia](Zhar_Ptytsia.md). It levels up as Ebony Wings does and benefits from Limit Breaking. Its bombardment area is slightly further than Ebony Wings\'s.
-  [Twilight Requiem (II)](Twilight_Requiem_(II).md) - Allows Ebony Wing\'s projectiles to explode when they expire. Damage is based on [Curse](Curse.md).
-  [Pale Diamond Incursion (V)](Pale_Diamond_Incursion_(V).md) - Bombarding zone now locks on nearest enemy. Gains  [Cooldown](Cooldown.md) bonus based on  [Speed](Speed.md).

## Levels {#Levels}

Ebony Wings has 8 levels. At max level, Ebony Wings has +20 [Base Damage](Base_Damage.md), +120% [Area](Area.md), +7 [Amount](Amount.md), and -0.5 seconds of [Cooldown](Cooldown.md).

|  |  |
|----|----|
| Level | Description |
| Level 1 | Bombards in a circling zone. |
| Level 2 | Fires 1 more projectile. Base Area up by 40%. |
| Level 3 | Fires 1 more projectile. Base Damage up by 10. |
| Level 4 | Fires 1 more projectile. Cooldown reduced by 0.3 seconds.^[\[1\]](#cite_note-cooldown-1)^ |
| Level 5 | Fires 1 more projectile. Base Area up by 40%. |
| Level 6 | Fires 1 more projectile. Base Damage up by 10. |
| Level 7 | Fires 1 more projectile. Cooldown reduced by 0.3.^[\[1\]](#cite_note-cooldown-1)^ |
| Level 8 | Fires 1 more projectile. Base Area up by 40%. |
