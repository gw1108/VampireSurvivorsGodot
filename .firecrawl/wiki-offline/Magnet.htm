Magnet

Magnet is one of the player stats in Vampire Survivors. It determines the radius inside which  Experience Gems and other pickups will be collected by the player's character.

The base value of Magnet is 30. It is only shown after starting the game, and not in the Character Selection menu. Magnet modifications stack multiplictively.

There is no upper limit to how much Magnet the player may have.

 Blood Astronomia (XXI) turns the character's Magnet area into a zone to damage enemies. It has a base damage of 5, multiplied by  Amount. It has a Cooldown of 2 seconds and no Knockback.

Magnet also affects Base Damage, Speed, Area and Duration of  Vol Umbra .

The Magnet PowerUp increases Magnet exponentially, multiplying it by 1.25 per rank, with a maximum rank of 2. At rank 2, it multiplies Magnet by 1.5625. It initially costs 300.

Rank Magnet
1 1.25x
2 1.5625x

The calculation of initial Magnet is done differently from other stats, similarly to that of  Max Health:

Magnet=30*Character bonus*PowerUp multiplier

For example,  Poe starts with 1.25x Magnet, so using Poe with the PowerUp at rank 2, Magnet would be:

30*1.25*1.5625=58.59375

From items

[edit | edit source]

Magnet's associated passive item is the  Attractorb, which provides varying multiplicative bonuses to Magnet, up to a maximum of 3.98x Magnet at level 5.

 Golden Egg can permanently increases Magnet base value by 0.3 for the character that collected it. So the calculation will be:

Magnet=(30+Eggs bonus)*Character bonus*PowerUp multiplier

Notes

[edit | edit source]
  • In the Character Selection menu the displayed Magnet value only shows the percentage applied at the start of the game instead of the actual value. Also, the shown percentage doesn't increase exponentially, which is incorrect. These two factors combined make it difficult to tell how much Magnet the player actually has at the start.
  • The Magnet value shown in the stats panel, both in the Character Selection menu and in-game, is rounded to the closest whole number. The actual value of Magnet is a decimal number.
  • To calculate Magnet initial in-game value using the value shown in the Character Selection menu is a little difficult as each egg bonus adds 1% to it, which is an additive bonus, not multiplicative. To do it, the following formula can be used: Magnet=(((Menu%Value−Character%Value−PowerUp%Value)*0.3)+30)*Character bonus*PowerUp multiplier
    • For example, Poe with no Eggs Bonus: Magnet=(((75−25−50)*0.3)+30)*1.25*1.5625=58.59375
      • Note that 0.3 is the value added by the Golden Egg in the Magnet base value, so the purpose of the formula is to find the bonus acquired with Golden Eggs using the %  showed in the menu as each one increases it by 1%, to sum it with the base value as showed in the previous formula and then to multiplying by the character and PowerUp bonuses.

Tips

[edit | edit source]
  • Bonuses to Magnet apply multiplicatively, meaning each bonus will provide a percent of the current total, not the original total. This means that the Attractorb, PowerUp, and playing as Poe provide more powerful bonuses as the stat gets increased.
    • With no bonuses to Magnet, it will go from 30, to 45, to 60, to 75, to 119.
    • With two levels into the Magnet PowerUp (which add an additional +50% to the stat), it will start at 47 and increase to 70, 94, 117, 140, and finally 187. The reason it starts at 47 and not 45 is because each +25% bonus from the PowerUp is applied separately, first taking 25% of 30 to give 37.5 and then 25% of 37.5 to give 47. Rounding is involved in the process.
    • For gold farming purposes an initial value of about 120 with a fully upgraded Magnet bringing it to 478 is required to pull gold bags from off screen, but for it to be effective the player will need slightly more.
    • With 2 levels of Magnet and Peppino's +100% bonus, Magnet will cap out at 373 which is around 4 times larger than the default pickup range and 120 points higher than other characters' reach.
    • Poe's range 1.56 times larger than default range and about 30 points higher.
