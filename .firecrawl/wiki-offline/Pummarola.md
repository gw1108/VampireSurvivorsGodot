**Pummarola** is a [passive item](Passive_item.md) in *[Vampire Survivors](Vampire_Survivors.md)*. It is unlocked by surviving 5 minutes as  [Gennaro Belpaese](Gennaro_Belpaese.md).

Pummarola increases  [Recovery](Recovery.md) by 0.2 per level.

It can be found as a [stage item](Stage_item.md) in [Mad Forest](Mad_Forest.md), [Moongolow](Moongolow.md), [Boss Rash](Boss_Rash.md), and [Mt.Moonspell](Mt.Moonspell.md).

## Levels {#Levels}

Pummarola has 5 levels. At max level, Pummarola provides +1  [Recovery](Recovery.md).

|         |                                                 |
|---------|-------------------------------------------------|
| Level   | Description                                     |
| Level 1 | \"Character recovers 0.2 HP per second.\"       |
| Level 2 | Health recovery increases by 0.2 HP per second. |
| Level 3 | Health recovery increases by 0.2 HP per second. |
| Level 4 | Health recovery increases by 0.2 HP per second. |
| Level 5 | Health recovery increases by 0.2 HP per second. |
