# Main menu {#firstHeading .firstHeading .mw-first-heading}

<figure typeof="mw:File/Thumb">

</figure>

The **main menu** is one of the several menus in *[Vampire Survivors](Vampire_Survivors.md)*. Once the player loads the game, a flashing \"PRESS TO START\" prompt will appear. When clicked, the player will gain access to the main menu.

This menu showcases a variety of options that are available from the start of the game, such as the [unlocks](Unlocks.md) and [collection](Collection.md) menus. However, some options, such as the [Bestiary](Bestiary.md) and the [Secrets](Secrets.md) menus, will only become visible by doing certain tasks. In many cases, these tasks would be to obtain certain [relics](Relics.md), such as the  [Ars Gouda](Ars_Gouda.md) and the  [Forbidden Scrolls of Morbane](Forbidden_Scrolls_of_Morbane.md), respectively, but for Quick Start, the player has to unlock all of the five main stages.

## Options {#Options}

<figcaption>The main menu encountered by a mobile player who has obtained all of the <a href="Relic.md">relics</a> needed to unlock the rest of the options.</figcaption>

**Note:** The following table lists all of the main menu options below the top navigation bar, from top to bottom, left to right, then lists the options in the top navigation bar.

|  |  |  |
|----|----|----|
| Option | Description | Unlocking |
| [DLC Store](DLC_Store.md) (mobile only) | Accesses the DLC Store, which lists all of the [expansions](Expansions.md). | [**?** (edit)](Main_menu.md) |
| Start | Accesses the [character selection menu](Character.md). | Available by default. |
| Quick Start | Immediately accesses a random stage with a randomly selected character. | Unlocking the five main stages of the game: [Mad Forest](Mad_Forest.md), [Inlaid Library](Inlaid_Library.md), [Dairy Plant](Dairy_Plant.md), [Gallo Tower](Gallo_Tower.md), and [Cappella Magna](Cappella_Magna.md). |
| Online | Accesses [online mode](Online_mode.md). | Available by default on all platforms, excluding iOS and Android devices. However, players on mobile devices can access this mode, as long as they have bought at least one paid [DLC](DLC.md) and have enabled [landscape mode](Landscape_mode.md) in their [options](Options.md). |
| [Adventures](Adventures.md) | Accesses the adventure selection menu. | Obtaining the  [Atlas Gate](Atlas_Gate.md) from [Boss Rash](Boss_Rash.md). |
| Credits | Lists everyone who has contributed to the development and growth of the game. | Available by default. |
| [Bestiary](Ars_Gouda.md) | Lists all of the [enemies](Enemies.md) encountered by the player so far. The bestiary entries include a description, stats, and other information about each enemy in the game. | Obtaining the  [Ars Gouda](Ars_Gouda.md) from [Dairy Plant](Dairy_Plant.md). |
| [Collection](Collection.md) | Lists all of the items encountered by the player so far. The types of items listed in the collection include [weapons](Weapons.md), [passives](Passives.md), [pickups](Pickups.md), [relics](Relics.md), and [arcanas](Arcanas.md). | Available by default. |
| [Power Up](PowerUps.md) | A shop where players can purchase various [PowerUps](PowerUps.md) in the game. These PowerUps grant permanent stat improvements to all [characters](Characters.md). | Available by default. |
| [Unlocks](Achievements.md) | A menu that lists various tasks that players can accomplish to unlock new items, characters, and more. | Available by default. |
| [Secrets](Secrets.md) | A menu that allows players to type in hints to instantly unlock items, characters, and more. | Obtaining the  [Forbidden Scrolls of Morbane](Forbidden_Scrolls_of_Morbane.md) from [The Bone Zone](The_Bone_Zone.md). |
