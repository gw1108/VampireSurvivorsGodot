**Attractorb** is a [passive item](Passive_item.md) in *[Vampire Survivors](Vampire_Survivors.md)*. It is unlocked upon finding a [Vacuum](Vacuum.md) dropped by a [light source](Light_source.md).

Attractorb gives multiplicative bonuses of varying size to the player\'s [Magnet](Magnet.md) stat.

Attractorb is required for the [evolution](Evolution.md) of [Santa Water](Santa_Water.md), [La Borra](La_Borra.md). It is also required for the evolution of [Mirage Robe](Mirage_Robe.md), [J\'Odore](J'Odore.md), in which Attractorb must also be fully leveled.

Attractorb can be found as a [stage item](Stage_item.md) in [Dairy Plant](Dairy_Plant.md), [Moongolow](Moongolow.md), [Boss Rash](Boss_Rash.md), [Tiny Bridge](Tiny_Bridge.md), [Mt.Moonspell](Mt.Moonspell.md), and [Abyss Foscari](Abyss_Foscari.md).

## Levels {#Levels}

Attractorb has 5 levels. Unlike other passive items, each level provides a different sized bonus.

|       |                                                 |                  |
|-------|-------------------------------------------------|------------------|
| Level | Description                                     | Effect total     |
| 1     | \"Character picks up items from further away.\" | 1.5x Magnet      |
| 2     | Pickup range increased by 33%.                  | 1.995x Magnet    |
| 3     | Pickup range increased by 25%.                  | 2.49375x Magnet  |
| 4     | Pickup range increased by 20%.                  | 2.9925x Magnet   |
| 5     | Pickup range increased by 33%.                  | 3.980025x Magnet |
