**Duplicator** is a [passive item](Passive_item.md) in *[Vampire Survivors](Vampire_Survivors.md)*. It is unlocked by getting the [Magic Wand](Magic_Wand.md) to Level 7.

Duplicator increases [Amount](Amount.md) by +1 per level.

Duplicator is required for the [evolution](Evolution.md) of [Lightning Ring](Lightning_Ring.md), [Thunder Loop](Thunder_Loop.md). It is also required for the evolution of [Summon Night](Summon_Night.md), [Echo Night](Echo_Night.md), in which Duplicator must also be fully leveled.

Duplicator can be found as a [stage item](Stage_item.md) in [Cappella Magna](Cappella_Magna.md), [Moongolow](Moongolow.md), [Boss Rash](Boss_Rash.md), and [Mt.Moonspell](Mt.Moonspell.md).

## Levels {#Levels}

Duplicator has 2 levels. At max level, Duplicator provides +2 [Amount](Amount.md).

|         |                                    |
|---------|------------------------------------|
| Level   | Description                        |
| Level 1 | \"Weapons fire more projectiles.\" |
| Level 2 | Fires 1 more projectile.           |
