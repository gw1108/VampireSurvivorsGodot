# Run summary

After the Death_screen.md, there is a run summary screen. It displays the Results in the top as a header followed by two columns. The left column has:
{STAGE_NAME} - {DIFFICULTY} - Gold x {GOLD_MULTIPLIER DEPENDENT ON RUN MODE (default 1)}.
Survived:           {TIME}
Gold earned:        {GOLD EARNED}
Level reached:      {PLAYER LEVEL}
Enemies defeated:   {ENEMIES DEFEATED}

Then a table
Weapon--------------------|LV------------------|Damage-----------|Time-------------------|DPS-----------|
weapon icon + weapon name-|Level of the weapon-|Total damage-----|Time player had weapon-|DPS calculated|

The right column has:
Character Icon + Character Name.
Weapon icon, Weapon icon, etc.
weapon level, Weapon level, etc.

Passive item icon, etc.
Passive item level, etc.

Pickupable item icon, etc.
Pickup count, etc.
