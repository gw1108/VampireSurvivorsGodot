Main menu

From Vampire Survivors Wiki
Jump to navigation Jump to search

The main menu is one of the several menus in Vampire Survivors. Once the player loads the game, a flashing "PRESS TO START" prompt will appear. When clicked, the player will gain access to the main menu.

This menu showcases a variety of options that are available from the start of the game, such as the unlocks and collection menus. However, some options, such as the Bestiary and the Secrets menus, will only become visible by doing certain tasks. In many cases, these tasks would be to obtain certain relics, such as the  Ars Gouda and the  Forbidden Scrolls of Morbane, respectively, but for Quick Start, the player has to unlock all of the five main stages.

Options

[edit | edit source]

Note: The following table lists all of the main menu options below the top navigation bar, from top to bottom, left to right, then lists the options in the top navigation bar.

Option Description Unlocking
DLC Store (mobile only) Accesses the DLC Store, which lists all of the expansions. ? (edit)
Start Accesses the character selection menu. Available by default.
Quick Start
 Immediately accesses a random stage with a randomly selected character. Unlocking the five main stages of the game: Mad Forest, Inlaid Library, Dairy Plant, Gallo Tower, and Cappella Magna.
Online Accesses online mode. Available by default on all platforms, excluding iOS and Android devices. However, players on mobile devices can access this mode, as long as they have bought at least one paid DLC and have enabled landscape mode in their options.
Adventures Accesses the adventure selection menu. Obtaining the  Atlas Gate from Boss Rash.
Credits
 Lists everyone who has contributed to the development and growth of the game. Available by default.
Bestiary Lists all of the enemies encountered by the player so far. The bestiary entries include a description, stats, and other information about each enemy in the game. Obtaining the  Ars Gouda from Dairy Plant.
Collection Lists all of the items encountered by the player so far. The types of items listed in the collection include weapons, passives, pickups, relics, and arcanas. Available by default.
Power Up A shop where players can purchase various PowerUps in the game. These PowerUps grant permanent stat improvements to all characters. Available by default.
Unlocks A menu that lists various tasks that players can accomplish to unlock new items, characters, and more. Available by default.
Secrets A menu that allows players to type in hints to instantly unlock items, characters, and more. Obtaining the  Forbidden Scrolls of Morbane from The Bone Zone.

Trivia

[edit | edit source]
  • The track that briefly plays upon accessing the main menu is an unlisted track titled "Side A."
    • However, if the player has enabled the "Blast Processed" option, the Side B track will be played instead.
  • The only green buttons are the "ADVENTURES", "POWER UP", and "DLC STORE" buttons.
    • The "POWER UP" button is the only green button that is available from the start of the game.
  • The only two buttons in the main menu that have a lower height and are in lowercase are the "quick start" and the "credits" buttons.
    • The "credits" button is the only squished and lowercase button that is available from the start of the game.
