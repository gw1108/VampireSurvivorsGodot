# Banish {#firstHeading .firstHeading .mw-first-heading}

[Jump to navigation](#mw-head) [Jump to search](#searchInput)

<figure class="mw-halign-right" typeof="mw:File">

</figure>

**Banish** is one of the [stats](Player_stats.md) in *[Vampire Survivors](Vampire_Survivors.md)*. It affects the amount of times the player can remove an [item](Item.md) choice for the rest of the run.

Once an item is banished it can no longer be offered as a [level up](Level_up.md) option. If an owned item is banished or if a banished item is later picked up as a [Stage item](Passive_item.md#Stage_items), it will remain at its current rank, but can no longer be offered upon level ups or upgraded by [Treasure Chests](Treasure_Chest.md). Banished [weapons](Weapon.md) can be offered [Limit Break](Limit_Break.md) upgrades, once the player\'s build is fully upgraded if Limit Break is enabled.

There is no upper limit to how many Banishes the player may have.

The Banish [PowerUp](PowerUp.md) increases Banish by 2 per rank, with a maximum rank of 5 after unlocking all Banish [achievements](Achievements.md). With one rank unlocked it initially costs 100.

|      |        |
|------|--------|
| Rank | Banish |
| 1    | +2     |
| 2    | +4     |
| 3    | +6     |
| 4    | +8     |
| 5    | +10    |

Each rank of PowerUp is unlocked by these achievements:

- Filling 60 entries in the [Collection](Collection.md).
- Filling 70 entries in the Collection.
- Filling 80 entries in the Collection.
- Filling 90 entries in the Collection.
- Filling 100 entries in the Collection.

## From items {#From_items}

Banish does not have an associated [passive item](Passive_item.md).

 [Golden Egg](Golden_Egg.md) permanently increases Banish by 0.2 for the character that collected it.
